Apache ServiceMix is an open-source ESB based on Apache Karaf that allows you to deploy, manage and monitor integration flows and services.

It provides a convenient and easy-to-use runtime container to allow you to leverage the full power of other Apache projects like:

  • Apache Camel for routing messages
  • Apache CXF for implementing and consuming web services
  • Apache ActiveMQ for reliable messaging and clustering
